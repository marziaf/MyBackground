# MyBackground
Client-server application for video background replacement

Project presentation: https://bit.ly/374PVZI

## Setup
To compile java files, run `setup`:
```
cd path/to/MyBackground

chmod 755 setup
./setup
```
## Execution
To start client run `start-client`
```
cd path/to/MyBackground

./start-client
```
To start server
```
./start-server
```
